# Responsibility Chart (RACI)
 
RACI is an acronym for the four major headings in a responsibility chart:
 
| Term | Definition |
|---|---|
| **Responsible** | Individual or group actively working to achieve the task |
| **Accountable** | Individual or group responsible for the completion of a given phase or task |
| **Consulted** | Individuals who provide consulting input into the phase or task |
| **Informed** | Individuals who are informed of the progress of the phase or task |
